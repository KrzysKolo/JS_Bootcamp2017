Parser adresu URL

    Użycie funkcji getPage() powinno wyglądać następująco:

    // dla adresu np. http://localhost/test/?page=2
    getPage(); // zwraca 2
    // dla adresu np. http://localhost/test/
    getPage(); // zwraca null
    // dla adresu np. http://localhost/test/?page=nieliczba
    getPage(); // zwraca null
