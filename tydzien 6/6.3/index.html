Informacja

Klasa ExchangeRate pozwala na pobierania kursów walut udostępnianych przez API NBP.

Klasy dziedziczące GbpRate, UsdRate oraz EurRate pozwalają natomiast pobierać kursy poszczególnych walut

Więcej

ExchangeRate klasa podstawowa

Instancja klasy ExchangeRate przyjmuje następujące argumenty:

symbol waluty:
(wymagane) trzyliterowy kod waluty (standard ISO 4217), pisany małymi literami

data:
(opcjonalnie) w formacie rrrr-mm-dd
Jeśli data nie zostanie podana, pobrane zostaną dane dla ostatniego zanotowanego przez NBP kursu danej waluty. Podobnie, jeśli data zostanbie podana niepoprawnie.

Aby uzyskać dostęp do notowania należy na instancji klasy wywołać metodę getData(), a nastepnie wywołać funkcję callback która pozwoli na operowanie na obiekcie z kursem waluty.

funkcja callback metody getData() przekazuje obiekt o następujacych właściwościach:
{
currency: symbol waluty (string),
name: nazwa waluty w języku polskim (string),
rate: średni kurs (number),
date: data publikacji pobranego kursu średniego (string),
buy: cena kupna (number),
sell: cena sprzedaży (string),
sellbuydate: data publikacji pobranyh kursów kupna i sprzedaży (string),
}

Przykład użycia:


        let instance = new ExchangeRate("gbp", "2016-03-30");
        instance.getData(obj => console.log(obj));

        /// zaloguje:
        /// { buy: 5.3643,
        /// currency:"GBP",
        /// date:"2016-03-30",
        /// midRate:5.3974,
        /// name: "funt szterling",
        /// sell: 5.4727,
        /// sellbuydate: "2016-03-30"}

UsdRate i pozostałe klasy dziedziczące

Klasy GbpRate, UsdRate oraz EurRate dziedziczą z klasy ExchangeRate i działają analogicznie jak ona, z tą różnicą, że nie wymagają podania symbolu waluty. Jedynym argumentem jaki przyjmują ich instancje jest opcjonalna data.

Przykład użycia:


        let instance = new EurRate();
        instance.getData(obj => console.log(obj));

        /// zaloguje:
        /// { buy:4.1772,
        /// currency:"EUR",
        /// date:"2017-05-19",
        /// midRate:4.2069,
        /// name:"euro",
        /// sell:4.2616,
        /// sellbuydate:"2017-05-19"}
